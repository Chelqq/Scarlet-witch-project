Manual de Usuario - Interfaz de Control del Robot InMoov

1. Introducción

Este manual describe el uso de la Interfaz de Control del Robot InMoov, una aplicación web diseñada para controlar el robot humanoide de código abierto InMoov.

2. Descripción del Robot

InMoov es un robot humanoide de tamaño real, diseñado para ser impreso en 3D y animado. Creado por Gael Langevin, este proyecto de código abierto facilita el acceso a la robótica avanzada.

3. Características Principales

3.1 Control de Movimiento

  • Movimientos Articulados de la Mano: Control de cada dedo mediante sliders.
  • Posicionamiento del Brazo: Manejo del hombro, codo y muñeca a través de cinemática directa.
  • Control de Cabeza y Cuello: Ajuste intuitivo de la mirada mediante controles de inclinación.
  • Biblioteca de Poses: Guardado y carga de posiciones comunes para acceso rápido.
  • Programación de Secuencias: Creación, guardado y reproducción de secuencias de movimiento.

4. Configuración Técnica

La aplicación usa un backend basado en Flask (Python), que se comunica con los sistemas de control del robot mediante conexiones seriales y protocolos personalizados.

5. Guía de Uso

  1. Acceder a la aplicación ingresando la dirección IP del servidor en un navegador compatible.
  2. Iniciar sesión con credenciales de usuario (o acceder como invitado con funciones limitadas).
  3. Navegar por los paneles de control a través del menú.
  4. Utilizar los controles deslizantes y botones en la sección "Controles Rápidos" para ejecutar movimientos básicos.
  5. Guardar configuraciones y secuencias de movimiento para su reutilización.

6. Aplicaciones Educativas

  • Explorar principios de robótica mediante interacción directa con el robot.
  • Comprender sistemas mecánicos a través del control de movimientos.
  • Aprender programación creando secuencias de movimiento personalizadas.
  • Desarrollar habilidades de resolución de problemas en robótica.

7. Desarrollo Futuro

  • Implementación de aprendizaje automático para mejorar los movimientos del robot.
  • Mejoras en visión por computadora para reconocimiento avanzado de objetos.
